BattleSim
=========
=========
A zero player battle simulation game with two teams that simulates itself where user can define map, units and implement their battle logic.
Features
--------
- Unit Creation: Create units with customizable attributes such as health and attack.
- Simple Battle Scripting: Define unit behavior using a straightforward scripting language.
- Team Dynamics: Units can belong to different teams and interact based on their scripts.
- Simulation Engine: Watch the battle unfold as units execute their scripts.
